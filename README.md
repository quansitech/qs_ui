### 兼容
目前该仓库下的组件只支持微信小程序与支付宝小程序
### 依赖的框架与库
该组件，依赖于[https://github.com/dcloudio/uni-app](uni-app)与[https://github.com/weilanwl/ColorUI](colorUI样式)。
使用前，需要安装ui-app,具体安装方法参考官方文档。colorUI的样式，可以在该项目的./src/static/styles的文件夹拷贝出你需要的样式。
